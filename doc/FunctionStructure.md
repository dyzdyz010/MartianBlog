MartianBlog功能架构
=================

前台
---

前台共有两个页面 —— `首页`，`文章详情`

1. 首页

	首页显示所有文章的完整列表。从上到下依次是：
	
	+ logo
	+ title
	+ subtitle
	+ article
		+ title
		+ date + tags
		+ content
		+ author
	+ footer
	
2. 文章详情

	文章详情页显示选定文章的详细内容，已经评论模块。从上到下依次是：
	
	+ logo
	+ title
	+ subtitle
	+ article
		+ title
		+ date + tags
		+ content
		+ author
	+ discussion
	+ footer
